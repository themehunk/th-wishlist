== Installation ==

1. Upload the `th-wishlist.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get started from Dashboard > TH Search

== Frequently Asked Questions ==

= How to Install Plugin =
Just upload the `th-wishlist.zip` to the `/wp-content/plugins/` directory, and then activate it from 'Plugins' menu in WordPress.

= How can i get Support if my TH Wishlist Plugin is not Working ? =

If you're facing any problem while using our TH Wishlist plugin, Please first follow the below steps :

1. Deactivate all plugins you are using and check if the problem is still occurring with our TH Wishlist.
2. Check the theme and plugin version, if it is latest or not.
3. Check the plugin with default theme, to make sure error is not due to theme you're using.

You're free to contact us through mail, via our Contact form and Live chat support. We have also given full documentation of our Plugin, you can go with it and learn how to use our plugin.

= How can i get more advanced features of TH Wishlist ? =

You can get more features with the premium version of TH Wishlist Plugin, available on [ThemeHunk](https://themehunk.com/). Please [follow this link](https://themehunk.com/th-wishlist/) and get to know about more advanced features.

= Is it Compatible with All WordPress Themes ? =

It is compatible with all [ThemeHunk themes](https://themehunk.com/free-themes/) and almost all WordPress themes, We always try to make our Plugin compatible with All WordPress Themes. If TH Advance Product Search will create any kind of issue with any theme. You're Free to [contact us](https://themehunk.com/contact-us/) , Our Team will be Happy to help you.
